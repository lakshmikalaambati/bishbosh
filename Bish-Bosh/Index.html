Bish-Bosh 2.0

Bish-Bosh 2.0 is simple webpage application

This application that will list all numbers from 1 to n(loopnumber) given by user with a few exceptions. If the number is evenly divisible by let us say X (first number given by user), it should print Bish. If the number is evenly divisible by let us say Y (secondnumber given by user) it should print Bosh. If the number is evenly divisible by both X and Y it should print out Bish-Bosh.
